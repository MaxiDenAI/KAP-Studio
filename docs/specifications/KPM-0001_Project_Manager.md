# KPM-0001 — Project Manager

## Назначение

Project Manager является стартовым экраном KAP Studio.

## Функции

- создание проекта;
- открытие `.kap`;
- список последних проектов;
- мастер Product DNA;
- выбор маркетплейсов;
- переход в редактор;
- хранение Recent Projects.

## Definition of Done

- приложение запускается без внешнего демо-файла;
- новый проект создаётся за один мастер;
- проект открывается двойным щелчком;
- последние проекты сохраняются локально;
- повреждённый проект показывает понятную ошибку.
