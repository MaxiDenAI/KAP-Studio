# KWE-0001 — KAP Workspace Engine

## Назначение

Workspace Engine управляет созданием, открытием, сохранением и версионированием файлов `.kap`.

## Обязательные функции

- создание контейнера;
- чтение manifest;
- сохранение атомарной заменой файла;
- увеличение revision;
- сохранение предыдущей версии в `history/`;
- экспорт в JSON;
- SHA-256 checksum.

## Ограничения

- бизнес-логика не размещается в UI;
- контейнер должен оставаться открытым и переносимым;
- сохранение не должно повреждать исходный файл.
