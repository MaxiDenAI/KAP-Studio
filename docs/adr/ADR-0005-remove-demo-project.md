# ADR-0005 — Отказ от встроенного демонстрационного проекта

**Статус:** Accepted

## Решение

KAP Studio больше не загружает встроенный `photoshop_2023.kap.json` при запуске.

Программа открывает Project Manager и предлагает:
- создать новый проект;
- открыть существующий `.kap`;
- выбрать последний проект.

## Причина

Встроенный демо-файл создавал зависимость от пути PyInstaller и мешал корректному запуску EXE.
