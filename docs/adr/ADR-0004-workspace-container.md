# ADR-0004 — Формат проекта `.kap`

**Статус:** Accepted  
**Версия:** 1.0

## Решение

Проект KAP хранится как ZIP-контейнер с расширением `.kap`.

## Состав

- `manifest.yaml`
- `project.json`
- `history/`
- `assets/`
- `exports/`
- `review/`

## Причины

- переносимость;
- открытый формат;
- поддержка истории;
- возможность будущего расширения;
- независимость от локальной базы данных.
